# Try This — Foder Test Prompts

Copy any of these prompts directly into foder and see what it does.

---

## Tips

- Use `@filename` to reference a file in your prompt: `@hello.py explain this code`
- Use `/pin hello.py` to keep a file in context for every message
- Use `/undo` if foder writes something wrong
- Use `/run` to auto-detect and run your project
- Use `!gcc -o app app.c && ./app` to compile and run C programs
- Use `/switch` to change to a faster or smarter model mid-session
